# IT 지식 위키 💻🚀

Obsidian으로 만드는 개인 IT 신기술·지식 위키입니다.

## 사용법
1. Obsidian 실행 → **Open folder as vault** → 이 `IT-note` 폴더 선택
2. 노트끼리는 `[[노트이름]]` 으로 연결합니다 (위키링크)
3. 왼쪽 아래 **그래프 뷰** 아이콘으로 지식 지도를 확인하세요

## 폴더 구조
| 폴더          | 용도                |
| ----------- | ----------------- |
| `00-Index/` | 대문(MOC) — 위키의 시작점 |
| `01-개념/`    | IT 용어·기술 개념 낱개 노트 |
| `02-기술노트/`  | 읽은 문서·아티클·책·강의 정리 |
| `03-주제정리/`  | 여러 개념을 엮은 정리 글    |
| `04-뉴스/`    | IT 신기술 뉴스 아카이브    |
| `05-실습/`    | 기술을 직접 써보는 실습·튜토리얼 |
| `99-템플릿/`   | 새 노트를 만들 때 쓰는 양식  |

## 첫걸음
👉 [[대문]] 에서 시작하세요.

## 추천 Obsidian 설정
- **Settings → Core plugins → Templates** 켜기 → 템플릿 폴더를 `99-템플릿` 으로 지정
- **Settings → Files & Links → New link format** 을 `Shortest path` 로 설정하면 링크가 깔끔해집니다

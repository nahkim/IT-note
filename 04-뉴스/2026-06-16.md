# 📰 IT 신기술 뉴스 — 2026-06-16

## 🔐 개발 / 보안 (오늘의 핵심 — '실제 악용' 제로데이 러시)

- **Chrome V8 제로데이 `CVE-2026-11645` 실제 악용**: V8 JS 엔진의 **OOB 읽기/쓰기**(CVSS 8.8)로 조작된 웹페이지만으로 브라우저 샌드박스 내 코드 실행. **올해 다섯 번째 Chrome 제로데이**. Chrome 149로 패치, CISA KEV 등재(6/9). ⚠️ **브라우저를 재시작해야** 패치가 적용된다. (V8 = JIT 엔진 → [[컴파일러와 인터프리터]])
- **LiteLLM(LLM 게이트웨이) 취약점 체인, 최대 CVSS ~9.9–10.0 · 실제 악용**: 저권한 사용자가 **권한 우회 → 프록시 admin 승격 → 서버 RCE**까지 연쇄(대표 CVE `CVE-2026-47101/47102/40217`, 별도 `CVE-2026-42271`도 CISA KEV). **AI 서빙 인프라 자체가 표적**이 된 사례 — 최신 stable로 업그레이드 권고. (→ [[LLM]] · [[AI 에이전트]] · [[인증과 인가]])
- **Palo Alto PAN-OS GlobalProtect 인증 우회 `CVE-2026-0257`(CVSS 7.8) 실제 악용**: 무단으로 **VPN 세션 생성** 가능. 초기 악용 정황 5/17, 6월 들어 경보 확대. (→ [[인증과 인가]])

## 🤖 AI / LLM

- **Fable 5 · Mythos 5 사태 심화 — 핵심은 '중국 접근 우려'**: David Sacks(6/13)는 "행정부가 Anthropic에 **'탈옥 수정 vs 모델 철수' 택일**을 제시했으나 Anthropic이 거부, 안전보다 소비자 모델 유지를 우선했다"고 주장. **Semafor·Fortune**은 **중국 연계 그룹의 Mythos 접근 정황**과 **Amazon의 경고**가 백악관 결정을 이끌었다고 보도. 두 모델은 여전히 중단 상태. (Anthropic은 "좁은 탈옥"이라 반박 — 양측 입장이 갈림. → [[2026-06-14]])
- **차기 플래그십은 계속 대기**: Google **Gemini 3.5 Pro**는 6월 출시 예고됐으나 **아직 미출시**(날짜 미정), xAI **Grok 5**도 지연 지속. 모델 출시 가속 흐름 자체는 이어짐. (→ [[2026-06-15]])

## ☁️ 클라우드 / 인프라

- **중국, $295B(약 2조 위안) 국가 AI 데이터센터 그리드 추진**: 5년간 전국 데이터센터망을 짓고 **칩·기술의 80%+를 국산(화웨이 등)**으로 조달. NDRC 청사진, China Mobile·China Telecom이 운영해 **2028년 단일 컴퓨팅 그리드**로 연결 — **엔비디아·AMD 의존 탈피** 겨냥. (미 수출통제와 맞물린 지정학 흐름)

# 📰 IT 신기술 뉴스 — 2026-06-25

## 🔐 개발 / 보안 (오늘의 핵심 — 'Cordyceps' 공급망 결함 + Cisco 통신서버 실악용)

- **'Cordyceps' — GitHub Actions를 노린 CI/CD 공급망 취약점 '계열' 공개**: 보안업체 Novee가 단일 CVE가 아니라 **CI/CD 워크플로 전반의 결함 계열**(명령 주입·인증 로직 우회·아티팩트 오염·권한 상승)을 묶어 'Cordyceps'로 명명. **고영향 저장소 약 3만 개를 스캔해 300개 이상이 완전 악용 가능**으로 확인됐고 Microsoft(Azure Sentinel)·Google(AI Agent Dev Kit)·Apache·Cloudflare·Python 재단 등에서 수정. 핵심은 **악성 풀리퀘스트(PR)의 제목·브랜치명·댓글** 같은 '신뢰되지 않은 입력'이 저권한 워크플로를 거쳐 **고권한 워크플로의 토큰**으로 흘러드는 '다단계 체인'이라 일반 스캐너가 놓친다는 점. 권고는 **워크플로 코드를 애플리케이션 코드만큼 엄격히 다루고 신뢰 경계를 강제**하라는 것. (→ [[CI-CD]] · [[인증과 인가]] · [[토큰]])
- **Cisco Unified CM SSRF `CVE-2026-20230` 실악용 격화(연속)**: 통합통신서버(Unified CM)·SME의 SSRF→root 결함(CVSS **8.6**, Cisco 영향도 'Critical')이 **6/24 Tor 경유 자동 웹셸 살포 + PoC 공개**로 위험이 급상승한 데 이어 경계 지속. WebDialer 활성 시스템은 즉시 패치(14SU6 등) — 공격 체인·악용 타임라인 상세는 → [[2026-06-24]]. (→ [[인증과 인가]] · [[HTTP와 HTTPS]])

## 🤖 AI / LLM (Fable 5·Mythos 5 — '복구 가능성'으로 국면 전환)

- **Trump "Anthropic은 이제 국가안보 위협 아니다" — 복구 기대 부상**: 6/12 수출통제로 전 세계 중단된 [[2026-06-21|Fable 5·Mythos 5 교착]]이 새 국면. 6/19 Axios 인터뷰에서 Trump는 **"지금은 아니다(국가안보 위협). 일주일 전이라면 모를까"**라며 Anthropic이 **"매우 책임감 있게 행동했다"**고 평가. Anthropic은 협상 대표를 **공동창업자·최고컴퓨트책임자 Tom Brown**으로 교체(정책총괄 Sarah Heck 동행)해 **'정치 분쟁이 아닌 기술 사안'** 프레임으로 상무부와 협의 중. 예측시장(Kalshi)은 **7/1 전 복구 ~57%**로, 일부 보도는 Fable 5가 앱 모델 선택기에 잠시 다시 노출됐다고 전함. 다만 **공식 복구 발표는 아직 없음**. (→ [[LLM]] · [[프롬프트 엔지니어링|탈옥]])
- **상무부 6/26 의회 서면답변 마감이 분수령**: Lutnick 상무장관은 **의원 4인에게 6/26까지 서면 답변**을 내야 함 — 모델 수정·사용 제한·모니터링 등 **복구 조건의 윤곽이 드러날 수 있는 첫 공식 문서**. 발단인 'Mythos 탈옥→사이버공격 능력' 주장과 Anthropic의 "좁은 단일 사례일 뿐"이라는 반박은 여전히 평행선. (→ [[AI 에이전트]])
- **Anthropic '인재 블랙홀' 계속 — 노벨상 John Jumper 합류**: AlphaFold 공동개발로 2024 노벨화학상을 받은 **John Jumper(Google DeepMind VP)**가 9년 만에 DeepMind를 떠나 Anthropic 합류를 6/19~20 발표. 올해 OpenAI 창립멤버 Andrej Karpathy(사전학습팀)·Azure AI 사장 Eric Boyd 등에 이은 영입 행렬로, [[2026-06-17|6/1 비공개 IPO 신청]] 흐름과 맞물려 경쟁이 가열. (→ [[Transformer]])

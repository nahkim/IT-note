# 📰 IT 신기술 뉴스 — 2026-07-23

## 🤖 AI / LLM (오늘의 핵심 — 3조 파라미터급 오픈웨이트 카운트다운)

- **Moonshot `Kimi K3` 오픈웨이트 D-4 — 7/27 공개 예정**: 7/16 공개된 **2.8조 파라미터** 프런티어 모델의 **가중치가 7/27 Modified MIT 라이선스로 풀린다**. **3조 파라미터 클래스 최초의 오픈웨이트**로, 새 어텐션 구조 **KDA(Kimi Delta Attention)**로 디코딩을 **최대 6.3배 가속**하고 **1M 토큰 컨텍스트**·**네이티브 비전**을 지원. 가격은 **입력 $3 / 출력 $15(M토큰)**로 공격적. 자체 벤치에서 Claude Opus 4.8·GPT‑5.5는 앞서지만 **Claude Fable 5·GPT‑5.6 Sol에는 뒤진다**고 밝힘. (→ [[LLM]] · [[토큰]] · [[임베딩]] · [[2026-07-22]])
- **프런티어는 여전히 폐쇄형 우위 — Anthropic `Claude Fable 5`**: Mythos급 프런티어 모델로 **현재 공개된 가장 똑똑한 LLM**으로 평가받으며, 벤치마크 큰 도약 대신 **Opus 계열의 약 2배 비용**. 오픈(Kimi K3)이 성능 격차를 좁히는 사이, 최상위 지능은 폐쇄형이 지킨다는 구도. (→ [[프롬프트 엔지니어링]] · [[RAG]] · [[벡터DB]])
- **출시 러시 지속 — 올해만 334+ 모델 추적**: GPT‑5.6 Sol(7/9)·구글 Gemini 경량 3종(7/21, [[2026-07-22]] 참고)에 이어 Kimi K3 가중치 공개가 임박하며 **경량·저가·오픈웨이트** 축의 경쟁이 계속 가열. (→ [[AI 에이전트]] · [[파인튜닝]])

## ☁️ 클라우드 / 인프라 (오늘의 핵심 — Blackwell 상용화와 'AI 슈퍼팩토리')

- **AWS EC2 `G7` 정식 출시 — RTX PRO 4500 Blackwell 첫 클라우드 탑재**: **NVIDIA RTX PRO 4500 Blackwell Server Edition GPU**를 얹은 G7 인스턴스가 **6/18~19 GA**(US East 오하이오·US West 오리건). 주요 클라우드 중 **최초 지원**으로, 전작 G6 대비 **AI 추론 최대 4.6배·그래픽 2.1배**. **GPU 8장·총 256GB 메모리**, 온디맨드 **g7.2xlarge $2.52/시 ~ g7.48xlarge $28.51/시**. (→ [[GPU vs CPU]] · [[nvidia-smi]] · [[수평 확장]])
- **AWS, GPU 예약 가격 20%↑(7/1 시행)**: EC2 **Capacity Block**(ML GPU 예약) 요금을 **약 20% 인상** — 공급·수요 압박이 이유. AWS는 **2026년 AI 인프라에 약 $200B capex**를 집행하고 **2027년 말까지 NVIDIA GPU 100만 개** 도입을 예고하는 등 물량·비용이 동시에 뛰는 국면. (→ [[캐시]] · [[GPU vs CPU]])
- **MS Azure 'AI 슈퍼팩토리' — Fairwater(위스콘신↔애틀랜타) 연결**: 두 Fairwater 사이트를 전용망으로 묶어 **행성 규모 AI 슈퍼팩토리**를 구성. **단일 평면 네트워크**로 수십만 장의 **NVIDIA GB200/GB300**을 하나의 슈퍼컴처럼 묶고, 랙당 **72 GPU(NVLink)·약 140kW**. 학습 잡을 '수개월→수주'로 단축한다는 설계. (→ [[로드 밸런서]] · [[마이크로서비스]] · [[온프레미스]])

## 🔐 개발 / 보안 (오늘의 핵심 — 역대 최대 패치와 계속 뚫리는 SharePoint)

- **SharePoint `CVE-2026-50522`(RCE, CVSS 9.8) 실악용 — 머신키 탈취**: 신뢰되지 않은 데이터 **역직렬화**로 미인증 원격 코드 실행이 가능. **7/20 유효 PoC 공개 직후 수 시간 내 실악용**이 관측됐고(watchTowr 허니팟), 공격자는 **머신키를 훔쳐 장기 지속성**을 확보. **7/14 패치** 완료 — 노출된 온프레미스 SharePoint는 즉시 패치 후 **머신키 로테이션** 필수. 지난 한 달 SharePoint 실악용의 **네 번째** 사례. (→ [[OWASP Top 10]] · [[인증과 인가]] · [[암호화와 해싱]] · [[온프레미스]])
- **역대 최대 7월 Patch Tuesday — CVE 약 570~620건·제로데이 3건**: 집계 편차는 있으나 **월간 사상 최대** 규모로, **제로데이 3건(실악용 2·공개 1)**을 수정. 그중 **`CVE-2026-56155`(AD FS 권한 상승)**는 **실악용 제로데이** — MS가 실제 공격 조사 중 발견했으며, ADFS 인증 특성상 **초기 발판 확보 후 측면이동→도메인 장악**으로 이어질 위험. (→ [[인증과 인가]] · [[OIDC]] · [[OAuth]])
- **SharePoint 실악용 파상공세 지속 — CISA KEV 누적 13건**: CISA는 `CVE-2026-32201`·`CVE-2026-45659`·`CVE-2026-56164`·`CVE-2026-58644` 등 다수 SharePoint 결함의 악용을 경고했고, **KEV의 SharePoint 항목이 누적 13건(올해만 5건 추가)**으로 [[2026-07-22]]의 엣지 장비 실악용 흐름과 함께 온프레미스 노출 자산이 계속 표적. (→ [[리버스 프록시]] · [[OWASP Top 10]] · [[쿠키 세션 JWT]])

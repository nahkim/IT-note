## 🔐 개발 / 보안 (오늘의 핵심 — SharePoint 랜섬웨어)

- **SharePoint RCE `CVE-2026-45659` — Storm-2603의 'Warlock' 랜섬웨어 실악용, 연방 패치 마감 7/4**: [[2026-07-03|어제 KEV 등재된]] 역직렬화 RCE(CVSS **8.8**)를 위협 그룹 **Storm-2603**이 **Warlock 랜섬웨어** 배포에 악용 중. **Site Member 권한만 있어도** 원격 코드 실행이 가능해 위험이 크고, Cloudflare 터널·Zoho Assist·VS Code SSH로 지속성을 확보한다. Microsoft 5월 패치분(Subscription Edition·2019·2016)이며 **연방기관(FCEB) 패치 시한이 오늘 7/4** — 온프레미스([[온프레미스]]) 서버는 즉시 패치·격리 권고. (→ [[웹사이트 보안]] · [[unpack]] · [[OWASP Top 10]])

## 🤖 AI / LLM

- **Anthropic, 중국의 '우회 접근' 차단 — Alibaba는 Claude Code 사내 금지로 맞대응**: FT 보도(7/3)에 따르면 Anthropic이 중국 기업들이 **싱가포르 자회사·VPN·해외법인(Azure 등)**을 통해 Claude에 접근하던 **우회로를 차단**. Ant는 싱가포르 법인 연동 계정, ByteDance는 VPN 개인 구독 환급 등이 지목됐고, Anthropic은 **타임존·사용 패턴으로 '중계 계정'을 탐지**하겠다고. 4~6월 **약 25,000개 부정 계정·2,880만 건 대화**(Alibaba 계열로 추정)를 적발했다고 밝힘. Alibaba는 사내에서 **Claude Code 사용을 금지**하며 맞대응. (→ [[LLM]] · [[인증과 인가]])

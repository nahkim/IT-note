# 📰 IT 신기술 뉴스 — 2026-06-23

## 🔐 개발 / 보안 (오늘의 핵심 — Ubiquiti UniFi OS 'CVSS 10.0' 3종 + Lantronix, CISA KEV 등재)

- **Ubiquiti UniFi OS, 최고 등급(CVSS 10.0) 취약점 3종 실제 악용 → CISA KEV 등재**: 6/23 CISA가 UniFi OS 취약점 3건을 **악용 중**으로 보고 KEV(알려진 악용 취약점 목록)에 추가. **`CVE-2026-34908`**(부적절한 접근제어, **10.0**) — 미인증 공격자가 시스템 설정을 무단 변경, **`CVE-2026-34909`**(경로 순회/Path Traversal) — 시스템 파일 읽기·조작, **`CVE-2026-34910`**(입력 검증 미흡, **10.0**) — **명령 주입(RCE)**. 셋을 엮으면 **미인증 → 임의 명령 실행 → 장치 완전 장악**으로 이어짐. **UniFi OS Server 5.0.8 이상**으로 즉시 패치 권고, 미 연방기관 **6/26까지 조치** 의무. (→ [[인증과 인가]] · [[리버스 프록시]])
- **Lantronix EDS5000도 'root 명령 주입' `CVE-2025-67038`(CVSS 9.8) 함께 등재**: 같은 6/23 KEV 추가 건. 엣지 디바이스 서버의 **HTTP RPC 모듈이 로그인 실패 로그를 남길 때 username을 셸 명령에 그대로 이어붙여**(sanitize 누락) **root 권한 임의 명령**을 실행. 펌웨어 **2.1.0.0R3** 영향, **2.2.0.0R1**로 업그레이드 권고. 같은 날 Lantronix EDS5000 플러그인 관련 코드 주입 건도 KEV에 묶여 총 4건이 추가됨. (→ [[인증과 인가]] · [[암호화와 해싱]])

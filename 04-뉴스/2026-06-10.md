# 📰 IT 신기술 뉴스 — 2026-06-10

## 🤖 AI / LLM

- **새 모델 출시 가속**: 평균 **약 2일에 한 번꼴**로 신규 [[LLM]]이 공개되는 흐름. 최근엔 **NVIDIA Nemotron 3 Ultra(550B)**, **Google Gemma 4 12B**, **Alibaba Qwen3.7 Plus** 등이 등장.
- **Google Gemini 3.5 Flash**: 5월 Google I/O(5/19)에서 공개. **플래그십급 성능을 Flash 속도로** 내면서 타 프런티어 모델 대비 **절반 이하 비용**, 에이전트·코딩 벤치마크에서 우위. 개인 [[AI 에이전트]] **Gemini Spark**(Gemini 3.5 기반)와 에이전트 우선 개발 플랫폼 **Google Antigravity**도 함께 발표(I/O에서 100여 개 발표).
- **Anthropic 급성장**: **Claude** 매출 런레이트 **$30B**(1분기 80배 성장; 5월 말엔 ~$47B로 추가 상승). 에이전트형 코딩 도구 **Claude Code**는 출시 6개월 만에 ARR $1B 돌파, **2026년 2월 기준 런레이트 ~$2.5B**.
- **OpenAI 엔터프라이즈 확장**: Bain·McKinsey·Capgemini 등 19개 파트너와 'OpenAI Deployment Company' 출범 — 대기업의 핵심 업무에 AI 내재화 지원.

## 🔐 개발 / 보안

- **Microsoft Build 2026**: 보안이 기본 내장된 엔터프라이즈 에이전트 개발을 위한 **Agent 365 SDK** 정식 출시 — 관측성·접근제어·컴플라이언스를 개발 워크플로에 통합.
- **VS Code 공급망 보안 강화**: 확장 자동 업데이트에 **2시간 지연**을 적용해 악성 확장 배포 리스크 완화.
- **GitHub Copilot 과금 변경**: 6월 1일부터 요청 기반 → **사용량 기반(metered) 과금**으로 전환.
- **취약점-익스플로잇 간격 붕괴**: "취약점 공개에서 공격까지 수 주 → 수 분/초"로 축소되는 추세. Linux 커널 use-after-free(**CVE-2026-23111**, nf_tables)로 비권한 사용자가 root 권한 상승 및 컨테이너 탈출 가능 — 2월 5일 업스트림 패치 완료. **6월 8일 Exodus Intelligence가 완전한 익스플로잇 코드를 공개**해 즉각적 위협으로 격상(커널 업데이트·재부팅 권고).

# 📰 IT 신기술 뉴스 — 2026-06-22

## 🤖 AI / LLM (오늘의 핵심 — Sakana, '모델'이 아니라 '오케스트레이터'를 내놓다)

- **Sakana AI, 'Fugu·Fugu Ultra' 정식 출시(GA) — 프런티어 모델 없이 프런티어 성능**: 6/22 일본 Sakana AI가 멀티에이전트 오케스트레이션 시스템 **Fugu**와 상위 티어 **Fugu Ultra**를 정식 출시. 핵심은 자체 거대 모델을 새로 학습한 게 아니라, **여러 프런티어 모델을 'agent pool'로 묶고 입력에 따라 어떤 모델에 위임할지·어떻게 통신하고 결과를 합칠지 스스로 결정하는 오케스트레이션용 LLM**이라는 점. 사용자에겐 단일 API·단일 모델처럼 보임(단, 어떤 모델이 답을 냈는지는 표시되지 않음). 6/12 수출통제로 [[LLM|Fable 5]]가 막힌 공백을 정조준한 출시라는 평. (→ [[AI 에이전트]] · [[2026-06-21]])
- **벤치마크는 'Fable 5와 어깨를 나란히' 주장 — 단, 모두 자체 보고치**: Sakana 발표 기준 Fugu Ultra가 **SWE-Bench Pro 73.7**, **TerminalBench 2.1 82.1**, **GPQA-Diamond 95.5**로 GPT-5.5·Claude Opus 4.8·Gemini 3.1 Pro를 앞서고 Fable 5·Mythos Preview와 동급이라고 주장. **다만 이 수치들은 전부 Sakana 자체 측정치이며 제3자 검증 전**이고, 단일 모델이 아닌 '시스템 전체' 성능이라는 점에 유의. (→ [[프롬프트 엔지니어링]])
- **가격·지역 제한**: 구독은 **Standard $20 / Pro $100 / Max $200(월)**, API는 입력 **$5~30**·출력 **$30~45**(/100만 토큰)대. **EU·EEA는 GDPR 등 규제 대응 중이라 출시 제외**(일정 미공개) — '오케스트레이션 자체를 상품으로 파는' 모델의 규제 마찰을 보여줌. (→ [[API 게이트웨이]])

## 🔐 개발 / 보안 (Mattermost 인증 우회 + AI 에이전트 공격면)

- **Mattermost 인증 결함 `CVE-2026-6673` (6/22 공개)**: 협업 플랫폼 Mattermost의 **Atlassian Connect(Jira 연동) 'installed' 콜백 인증 누락**으로, **미인증** 원격 공격자가 설치 대기창 동안 `/ac/installed`에 POST해 **위조 sharedSecret을 주입 → Jira 연동을 교란**. 영향 버전 11.7.0 이하·11.6.2 이하·11.5.5 이하·10.11.17 이하, **11.8.0/11.7.1/11.6.3/11.5.6/10.11.18에서 수정**(CVSS **6.4** 중간, MMSA-2026-00654). (→ [[인증과 인가]] · [[REST API]])
- **'Agentjacking'(Sentry MCP 인젝션) 파장 지속**: 가짜 에러 리포트에 악성 markdown을 실어 Claude Code·Cursor·Codex 같은 [[AI 에이전트|코딩 에이전트]]를 탈취하는 공격(성공률 ~85%·노출 DSN 2,388곳·별도 CVE 미부여)이 주말 내내 보안 매체·권고로 확산. Sentry는 **"구조적으로 방어 불가"**라며 완화를 모델 벤더로 넘긴 상태 — 'MCP 신뢰경계'가 새 공급망 위협이라는 경고. (최초 공개 상세 → [[2026-06-14]] · [[프롬프트 엔지니어링|프롬프트 인젝션]])

## ☁️ 클라우드 / 인프라 (메모리 슈퍼사이클이 'AI 메모리세'로)

- **'AI 메모리세'가 내 PC 값까지 올린다 — Micron 마진이 신호**: 6/22 보도는 Micron의 **기록적 마진**(Q3 FY26 가이던스 비GAAP 매출총이익률 **~81%**, 과거 메모리 호황기 30~45% 대비 두 배 이상)을 근거로, **AI 인프라 비용이 소비자 전자제품 가격으로 전가되는 메커니즘**을 짚음. **2026년 HBM 물량은 이미 전량 선계약 매진**, 하이퍼스케일러 선납금 누적 **$220억** 규모로 공급 핍박이 2026년 이후까지 지속 전망. 6/19 다룬 **HBM 메모리 슈퍼사이클**의 수익화 단면. (※ Micron 분기 실적 발표 자체는 6/24 → [[2026-06-19]] · [[2026-06-23]])
- **인재 이동도 'AI 인프라 쟁탈전'의 일부 (이주 흐름)**: 6/18 Transformer 공동저자 **Noam Shazeer가 Google DeepMind→OpenAI**(아키텍처 리서치 총괄)로 이동, 이어 AlphaFold의 노벨화학상 수상자 **John Jumper가 Anthropic 합류** 보도까지 — 모델 아키텍처·인프라 두뇌를 둘러싼 빅테크 간 쟁탈전이 6월 내내 이어짐. (→ [[Transformer]] · [[2026-06-21]])

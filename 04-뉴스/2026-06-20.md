# 📰 IT 신기술 뉴스 — 2026-06-20

## 🔐 개발 / 보안 (오늘의 핵심 — Splunk 미인증 RCE, 연방 패치 기한 6/21 임박)

- **Splunk Enterprise `CVE-2026-20253`(CVSS 9.8) — 미인증 원격 코드 실행, CISA KEV 등재·이번 주말이 마지노선**: Splunk가 함께 묶어 띄우는 **PostgreSQL 사이드카 서비스 엔드포인트에 인증 통제가 빠져** 있어, **네트워크만 닿으면 자격증명 없이** 서버의 **임의 파일 생성·절단(truncate)** 이 가능 → 이를 발판으로 **미인증 RCE**까지 이어짐. 연구자 허니팟에는 **6/16부터 악용 시도**가 잡혔고, **6/18 Splunk PSIRT가 "제한적 실제 악용"을 확인**, 같은 날 **CISA KEV에 등재**. 연방기관(FCEB) **패치 기한은 6/21** — 즉 주말 사이 조치가 핵심. **10.0.7·10.2.4·10.4.0** 이상으로 업그레이드 권고이며 **Splunk Cloud는 미영향**. (→ [[인증과 인가]] · [[관계형 데이터베이스와 SQL]] · [[2026-06-19]])
- **LiteLLM 'AI 게이트웨이' `CVE-2026-42271` — 체인 시 미인증 RCE(합산 10.0)**: 다수 LLM API 트래픽을 한 곳으로 모으는 **LiteLLM 프록시**의 MCP 미리보기용 엔드포인트(`/mcp-rest/test/...`)가 요청 본문의 **command·args·env를 그대로 받아 자식 프로세스로 실행** → **명령 주입**(CVSS 8.7). 여기에 Starlette **호스트 헤더 검증 우회 `CVE-2026-48710`** 를 엮으면 **인증 자체를 우회한 미인증 RCE**(합산 **10.0**)가 완성됨. **6/8 CISA KEV 등재**(실제 악용), **LiteLLM 1.83.7 + Starlette 1.0.1**로 패치. 성공 시 게이트웨이를 거치는 **모든 모델 제공사 API 키·사용 로그·하위 인프라**가 노출 — 'AI 인프라 자체가 새 공격면'이라는 신호. (→ [[API 게이트웨이]] · [[AI 에이전트]] · [[리버스 프록시]] · [[토큰]])

## 🤖 AI / LLM

- **Z.ai(전 Zhipu) GLM-5.2 가중치, MIT로 전면 공개 — '프런티어급 오픈웨이트 코딩 모델'**: 6/13 코딩 플랜 구독자에게 먼저 풀린 뒤 **6/16~17 모델 가중치를 MIT 라이선스로 공개**(Hugging Face). **약 750B 파라미터(MoE, 토큰당 ~40B 활성)** · **100만 토큰 컨텍스트**에, 1M 컨텍스트 비용을 잡기 위한 **IndexShare 희소 어텐션**을 더함. SWE-bench Pro 62.1·Terminal-Bench 2.1 81.0 등 **장기 코딩 벤치마크에서 폐쇄형과의 격차를 좁혔다**는 평가. 6월 들어 NVIDIA **Nemotron 3 Ultra**(6/4·550B), Moonshot **Kimi K2.7-Code**(6/12)로 이어진 **오픈웨이트 물결**의 최신 정점. (출시 당일 상세 → [[2026-06-13]] · [[LLM]] · [[파인튜닝]] · [[프롬프트 엔지니어링]])
- **AI 모델 출시 '집계 블로그' 주의**: 일부 매체의 'GPT-5.6·Gemini 3.2 정식 출시'류는 **출처별 상충**이라 본문 제외(현재 폐쇄형 최상위는 Claude **Opus 4.8**·GPT-5.5 라인업). Google **Gemini 3.5 Pro**도 **5/19 I/O 예고 이후 6월까지 모델 ID·날짜 미공개** 상태가 이어짐. (→ [[2026-06-19]])

## ☁️ 클라우드 / 인프라

- **메모리 슈퍼사이클, 이제 'PC·스마트폰' 지갑까지 — 데이터센터가 메모리를 빨아들인다**: AI 서버 우선 공급으로 메모리 제조 캐파가 고마진 제품으로 쏠리면서, TrendForce 기준 **1Q26 서버 DRAM 계약가 +90% 안팎(QoQ, 사상 최대)·NAND +55~60%**. 그 여파로 **PC·모바일·소비자용 DRAM 가격까지 동반 인상**, 일부 기업용 DDR5는 2025년 초 대비 **2배 이상**. **2026년 전 세계 메모리의 최대 ~70%를 데이터센터가 소비**, **HBM은 DRAM 웨이퍼의 ~23%** 점유 전망. 어제 다룬 'Rubin 랙 메모리 +435%·HBM 슈퍼사이클'이 **소비자 시장 비용으로 전이되는 국면**. (→ [[2026-06-19]] · [[수평 확장]])

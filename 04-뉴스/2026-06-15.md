# 📰 IT 신기술 뉴스 — 2026-06-15

## 🔐 개발 / 보안 (오늘의 핵심)

- **🚨 Verizon DBIR 2026 — '취약점 악용'이 자격증명 탈취를 제치고 침해 1위 경로**: 19년 역사상 처음으로 **[[인증과 인가|자격증명]] 탈취가 1위에서 밀려남**. **취약점 악용 31%** vs **자격증명 13%**. 22,000+ 침해 분석(전년 대비 2배). AI로 공격이 가속돼 **방어 시간이 '개월 → 시간'**으로 축소. CISA KEV 중 완전 패치율은 **26%**(전년 38%)로 하락, 중간 패치 소요는 43일로 증가.
- **ShinyHunters, Oracle PeopleSoft 제로데이(CVE-2026-35273) 악용 — 대학 직격**: [[HTTP와 HTTPS|HTTP]] 접근만으로 **무인증·무상호작용 RCE(CVSS 9.8)**. Mandiant(UNC6240) 추적상 **5/27~6/9** 활동, Oracle 권고는 **6/10**에야 나와 그전까지 완전한 제로데이. 통보된 **100+ 조직의 68%가 대학**, 탈취 데이터는 유출 사이트에 공개+협박. (PeopleTools 8.61/8.62)
- **Cisco SD-WAN, 올해 연속 제로데이**: 인증 우회로 admin 탈취되는 **CVE-2026-20182**(올해 6번째 악용, UAT-8616)에 이어 **CVE-2026-20245**(7번째·패치 미제공)까지. 같은 주 **Fortinet·Ivanti·SAP**도 크리티컬 패치(FortiSandbox 명령주입 CVE-2026-25089, CVSS 9.1). → 어제(에이전트 표적) 흐름과 합쳐 **"빠른 취약점 악용"이 2026 보안 키워드** ([[2026-06-14]])

# 📰 IT 신기술 뉴스 — 2026-06-24

## 🔐 개발 / 보안 (오늘의 핵심 — Cisco Unified CM 실제 악용 격화 + AI 툴체인 취약점 무더기)

- **Cisco Unified CM `CVE-2026-20230`(CVSS 8.6) — 'PoC 공개 → 실제 악용 → 웹셸 자동 배포'로 격화**: Unified Communications Manager·SME에서 **HTTP 요청 입력 검증 미흡 → 서버측 요청 위조(SSRF)**로 미인증 원격 공격자가 OS에 파일을 쓰고 **root 권한 상승**까지 노릴 수 있는 결함. 6/3 패치 공개 시점엔 악용 정황이 없었으나, **6/22 주말 Defused가 실제 악용을 첫 확인**(file:// 파일 쓰기 페이로드가 디코이에 착탄)했고, **6/24엔 Tor 경유 자동 스윕으로 웹셸을 떨구는 단계**로 진화 — `WebDialer` SSRF를 악용해 가짜 Apache Axis 서비스를 띄우고 **3단계 JSP 명령 셸**을 심는 방식. 웹셸이 박히면 **패치만으로는 공격자를 못 쫓아내므로**(별도 침해 점검 필요) 즉시 패치 + **`WebDialer` 비활성화**(기본 비활성)·관리 인터페이스 노출 최소화가 권고됨. (→ [[인증과 인가]] · [[리버스 프록시|SSRF]])
- **'AI 개발 툴체인' 취약점이 한날에 무더기 공개 — 패치율 0%**: 6/24 CVE 브리프 기준 **Flowise `CVE-2026-56274`(CVSS 9.9)**, **Crawl4AI `CVE-2026-53753`(9.8)**, **Langflow `CVE-2026-48519`(9.6)**, **picklescan `CVE-2026-56315`(9.8)** 등 AI/ML 빌더·크롤러·역직렬화 검사 도구에서 **원격코드실행·안전하지 않은 역직렬화** 결함이 동시 다발 공개. 공통점은 **공개 시점 벤더 패치 0%** — 신뢰할 수 없는 입력 차단·서비스 격리·벤더 릴리스 모니터링이 임시 방어선. '에이전트를 짜는 도구 자체'가 공격면이 되는 흐름. (→ [[AI 에이전트]] · [[JSON과 직렬화|역직렬화]])
- **실제 악용 목록도 갱신 — Splunk `CVE-2026-20253` 등 5건**: 6/24 기준 활발히 악용 중인 CVE로 **Splunk Enterprise `CVE-2026-20253`**(PostgreSQL 사이드카 미인증 임의 파일 쓰기 → RCE 연쇄·CISA KEV 등재), **Ubiquiti UniFi OS** 3종, **Lantronix EDS5000**이 거론됨. Splunk 건은 어제 흐름의 연장선. (→ [[2026-06-23]])

## 🤖 AI / LLM

- **영국 FCA "법은 결코 AI를 따라잡지 못한다" — 규제 패러다임 전환 선언**: Nikhil Rathi FCA 최고경영자가 6/24 'Agents of Change: Generative and Agentic AI in Financial Services 2026' 콘퍼런스에서 **"기술이 규제 패러다임보다 훨씬 빠르게 움직이고, 입법은 결코 따라잡지 못한다"**고 발언. **"일부 영역엔 여전히 세부 규칙이 필요하지만, 다른 영역에선 전통적 룰메이킹이 더는 통하지 않는다"**며 세칙 중심에서 **'관리·협력(stewardship)' 중심**으로의 전환을 예고. FCA는 시장남용 감시·대응 속도를 높이기 위해 **에이전트형 AI를 '초동 대응자(first responder)'**로 시험 중이며, 곧 Mills Review(AI가 소매 금융을 어떻게 바꾸는지)와 'AI 모범·미흡 사례' 발간을 예고. (→ [[AI 에이전트]] · [[LLM]])
- **모델 경쟁은 '대기' 기조 유지**: 6/24에도 빅랩의 차기 플래그십 정식 출시 소식은 없음 — Google **Gemini 3.5 Pro**(5/19 I/O 예고, 6월 현재 미출시), xAI **Grok 5**(지연 지속). 일부 집계 블로그의 'Gemini 3.1 Ultra·GPT-5.6·Gemini 3.2 정식 출시' 류는 출처별 상충·날짜 오기라 본문 제외. Meta–AMD $100B 칩 딜(2/24)·Karpathy의 Anthropic 합류(5/19)도 6/24 신규 뉴스가 아닌 기존 사건. (→ [[2026-06-23]])

## ☁️ 클라우드 / 인프라

- **Nebius, 'AI Cloud 3.6 Aether' 공개 — 콘솔에 내장된 인프라 운영 에이전트 'Echo'**: 6/24 Nebius가 **프로덕션 AI 운영**에 초점을 둔 3.6 릴리스를 발표(BusinessWire). 핵심은 **Echo** — 로그인 즉시 쓸 수 있는 **콘솔 내장 AI 에이전트**로, 내 인프라 맥락을 이해해 질문에 답하고 **가드레일을 둔 채 명령을 실행**(자연어로 클라우드 인프라 제어).

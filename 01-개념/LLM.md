# LLM (Large Language Model, 대규모 언어 모델)

## 한 줄 정의
방대한 텍스트로 학습해 다음 [[토큰]]을 예측하는, 수십억~수조 개 파라미터의 신경망 언어 모델.

## 자세히
- 대부분 [[Transformer]] 구조를 기반으로 한다.
- "다음 단어(토큰) 맞히기"라는 단순한 목표를 거대한 규모로 학습하면, 번역·요약·코딩·추론 같은 능력이 **창발적으로** 나타난다.
- 사전학습(pre-training) → [[파인튜닝]](fine-tuning) / RLHF 순서로 만들어진다.
- 한계: **환각(hallucination)**, 최신 정보 부재, 학습 데이터 편향.

## 왜 중요한가
- 검색·코딩·고객지원 등 거의 모든 소프트웨어 UX를 바꾸고 있다.
- 환각·최신성 한계는 [[RAG]]로 보완한다.

## 관련 개념
- [[Transformer]] — LLM의 기반 아키텍처
- [[RAG]] — 외부 지식으로 LLM을 보강하는 기법
- [[임베딩]] — 텍스트를 벡터로 바꿔 의미를 다룬다
- [[파인튜닝]] — 모델을 특정 목적에 맞추는 추가 학습
- [[AI 에이전트]] — LLM이 도구를 쓰며 일을 수행하는 형태

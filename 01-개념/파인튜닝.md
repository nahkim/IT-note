# 파인튜닝 (Fine-tuning, 미세조정)

## 한 줄 정의
이미 사전학습된 [[LLM]]을 **특정 데이터·작업에 맞춰 추가 학습**시켜 행동을 바꾸는 것.

## 자세히
- 사전학습(범용 지식 습득) → 파인튜닝(특정 목적 맞춤). 처음부터 학습하는 것보다 훨씬 적은 데이터·비용으로 가능.
- 종류:
  - **SFT(지도 미세조정)** — 입력·정답 쌍으로 학습.
  - **RLHF** — 인간 선호 피드백으로 답변 스타일을 정렬.
  - **LoRA / PEFT** — 일부 파라미터만 살짝 건드리는 **경량 파인튜닝**(저비용).
- **[[RAG]]와의 비교** (자주 헷갈리는 지점):
  - 파인튜닝 = 모델의 **말투·형식·기술**을 바꾼다(지식을 '체화').
  - RAG = 모델 밖의 **최신·사내 지식을 그때그때 검색**해 넣는다.
  - 실무에선 둘을 **함께** 쓰기도 한다.

## 왜 중요한가
- 범용 모델을 우리 도메인·말투·출력 형식에 맞추는 표준 방법.
- 단, 지식 최신화와 환각 보완은 보통 [[RAG]]가 더 싸고 빠르다 → "무엇을 바꾸려는가"로 선택.

## 관련 개념
- [[LLM]] — 파인튜닝의 대상
- [[RAG]] — 지식 보강의 대안이자 보완책
- [[토큰]] — 학습량·비용을 재는 단위

# FastAPI (패스트API)

## 한 줄 정의
**타입 힌트 기반**으로 빠르게 만드는 현대적인 파이썬 웹 API 프레임워크.

## 자세히
- **Starlette**(웹/ASGI) + **Pydantic**(데이터 검증) 위에 올라가 있다.
- 함수 매개변수에 타입 힌트만 붙이면 **요청 검증 + 자동 문서(Swagger/OpenAPI)**가 따라온다.
- **비동기(async/await)**를 지원해 I/O 위주 작업에서 높은 처리량을 낸다.
- ML 모델을 직접 API로 감쌀 때 가장 흔히 쓰이지만, **직렬화·배칭·도커라이즈·스케일링은 직접** 해야 한다 → 그 수고를 모델 서빙에 맞춰 표준화한 게 [[BentoML]].

## 왜 중요한가
- 파이썬 백엔드·ML 서빙의 사실상 표준 프레임워크 중 하나.
- [[BentoML]] 같은 상위 서빙 프레임워크도 내부적으로는 이런 ASGI 서버 위에서 돈다.

# 콜백 (Callback)

**한 줄 뜻**: 나중에 **특정 이벤트가 나면 시스템·라이브러리가 대신 불러주는 함수**. 내가 직접 호출하지 않고 "이 일이 생기면 이걸 실행해줘"라고 **함수를 넘겨두는** 방식.

**부연**:
- "내가 확인(pull, [[폴링]])"의 반대편 — **불려지는(push)** 쪽. 이벤트 기반·비동기의 기본 축([[동기와 비동기]]).
- 예: 버튼 클릭 핸들러, 타이머 콜백, **오디오 입력 콜백**(마이크 조각이 준비되면 호출).
- ⚠️ **실시간/고빈도 콜백은 빨리 반환해야 함** — 안에서 무거운 작업(디스크 I/O·락)을 하면 전체가 밀린다(예: 오디오 드롭아웃). → [[오디오 녹음 파이프라인]]

**관련**: [[동기와 비동기]] · [[폴링]] · [[오디오 녹음 파이프라인]]

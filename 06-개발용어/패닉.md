# 패닉 (Panic)

**한 줄 뜻**: 복구 불가능한 상황에서 **프로그램(또는 스레드)을 즉시 중단**시키는 것. Rust `panic!`, Go `panic`.

**부연**:
- **복구 가능한 오류 vs 패닉** — 예상되는 오류는 **값으로 반환**해 처리(Rust `Result`, Go `error`), 정말 회복 불가하거나 심각한 버그일 때만 패닉.
- Rust — 기본은 스택 **unwind**(또는 `abort`로 설정 가능), `std::panic::catch_unwind`로 잡을 수 있음. Go — `panic`을 `recover`로 방어.
- 원칙: **라이브러리 코드는 함부로 패닉하지 말고** 오류를 반환해 호출자가 결정하게 한다.

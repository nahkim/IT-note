# 레이스 컨디션 (Race Condition)

**한 줄 뜻**: 여러 스레드/프로세스가 **공유 자원에 동시에 접근**할 때, 실행 **타이밍·순서에 따라 결과가 달라지는** 버그. 적절한 동기화가 없을 때 생긴다.

**부연**:
- 전형 예: 두 스레드가 같은 카운터를 read → +1 → write 하면 한쪽 갱신이 **유실**.
- **비결정적**이라 재현·디버깅이 어려움 → [[하이젠버그]]·[[스퓨리어스 에러]]의 단골 원인.
- 방지: 락(뮤텍스)·원자적 연산·불변 공유·메시지 전달. 깊게는 → [[동시성 문제]]

**관련**: [[동시성 문제]] · [[하이젠버그]] · [[데드락]]

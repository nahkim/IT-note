# 이벨류레이션 (Evaluation)

**한 줄 뜻**: 식(expression)이나 코드를 **실행해 값을 구하는 것**. 넓게는 "평가" — 무언가를 계산·판정하는 과정.

**부연**:
- **식 평가** — `2 + 3 * 4`를 계산해 값 `14`를 얻는 것. `eval()`은 문자열을 코드로 평가(위험해서 주의).
- **eager(즉시) vs lazy(지연) evaluation**:
  - *eager* — 바인딩·전달되는 즉시 계산(call-by-value).
  - *lazy* — 값이 **필요할 때까지 미루고** 결과를 캐시(call-by-need). 불필요한 계산·무한 자료구조 처리에 유리. → [[함수형 프로그래밍]]
- **모델 평가(ML/LLM)** — 별개 뜻: 모델 성능을 지표로 측정(eval set). 이 맥락의 "evaluation"은 '성능 평가'.

# 워커라운드 (Workaround)

**한 줄 뜻**: 근본 원인을 고치지 않고 문제를 **임시로 우회·회피하는 해결책**. 정식 수정(fix)이 나오기 전까지의 임시 대응.

**부연**:
- **fix vs workaround** — fix는 원인을 직접 해결, workaround는 **원인은 그대로 두고 증상만 피해 감**.
- 어디까지나 임시라서, 방치하면 **기술 부채**로 쌓인다 → 나중에 제대로 fix 필요.
- 예: 버그 있는 API 대신 다른 경로로 우회, 특정 버전에서만 나는 오류를 조건 분기로 피하기.

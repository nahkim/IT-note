# 크로스플랫폼 (Cross-platform)

**한 줄 뜻**: **하나의 코드베이스로 여러 플랫폼**(iOS·Android·데스크톱 등)을 대상으로 만드는 방식. [[네이티브 앱]]의 트레이드오프 상대.

**부연**:
- 대표 프레임워크: **React Native · Flutter · Kotlin Multiplatform · .NET MAUI**, 데스크톱은 **[[Tauri]]·Electron**.
- 장점 — 개발·유지 **효율**(한 벌), 빠른 출시.
- 단점 — 네이티브 대비 **성능·플랫폼 밀착도**에서 타협, 최신 OS 기능 반영이 늦을 수 있음.

**관련**: [[네이티브 앱]] · [[Tauri]] · [[클라이언트 앱]]

---
> ✅ 교차검증 — cross-platform(단일 코드베이스로 다중 OS 타깃, React Native/Flutter/MAUI 등)과 네이티브 대비 장단점을 복수 자료로 확인.

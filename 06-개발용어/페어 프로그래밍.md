# 페어 프로그래밍 (Pair Programming)

**한 줄 뜻**: 두 개발자가 **한 화면에서 함께** 코딩하는 XP(익스트림 프로그래밍) 기법. **실시간 코드 리뷰**이자 협업.

**부연**:
- 역할: **드라이버(Driver)** = 키보드를 잡고 코드를 씀 · **내비게이터(Navigator)** = 방향·전략을 보며 실시간 리뷰·제안. 주기적으로 **교대**한다.
- 실시간판 [[코드 리뷰]] — 결함을 짜는 순간 잡고 지식도 즉시 전파. XP의 원조 12개 실천법 중 하나.
- 두 명 투입(비용) vs 품질·전파(이득)의 **트레이드오프**로 늘 논쟁 대상. 어려운 문제·온보딩에 특히 유효.

**관련**: [[코드 리뷰]] · [[적대적 리뷰]] · [[소프트웨어 테스트]]

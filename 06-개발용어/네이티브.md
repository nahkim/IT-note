# 네이티브 (Native)

**한 줄 뜻**: 특정 **플랫폼(OS)·하드웨어(CPU)에 맞춰 만들어져 그 위에서 직접 도는** 것. "덧대는 층 없이 그 환경 것으로."

**부연**:
- 두 축으로 쓰인다 — **[[네이티브 앱]]**(플랫폼 전용 앱)과 **[[네이티브 코드]]**(그 CPU용 기계어).
- "네이티브 지원" = 별도 우회/에뮬레이션 없이 **그 플랫폼이 기본 제공**(예: OS의 네이티브 오디오 API = [[오디오 캡처 API|WASAPI/Core Audio]]).
- 반대편: **[[크로스플랫폼]]**(여러 플랫폼 공용), 매니지드/바이트코드·인터프리터, 웹.

**관련**: [[네이티브 앱]] · [[네이티브 코드]] · [[크로스플랫폼]]

---
> ✅ 교차검증 — native = 특정 플랫폼/하드웨어 전용으로 직접 동작(앱·코드 두 축), 크로스플랫폼·매니지드의 반대 개념임을 복수 자료로 확인.

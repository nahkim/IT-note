# 📖 개발 용어 사전

개발하면서 마주치는 **용어·약어·표현**을 짧게 정리하는 글로서리입니다.
"이거 무슨 뜻이지?"를 **빠르게 찾아보는 용도**.

## `01-개념/`과의 차이
| | `01-개념/` | `06-개발용어/` |
|---|---|---|
| 성격 | 깊은 **개념 노트** | 짧은 **용어 사전** |
| 분량 | 원리·구조·"왜 중요한가"까지 | 한두 문단, 뜻 위주 |
| 목적 | 이해·학습 | 빠른 검색·복습 |

> 용어를 파다 내용이 길어지면 `01-개념/`으로 승격시켜도 됩니다. (그때는 `[[위키링크]]`로 연결)

## 작성 규칙
- 파일명: `용어.md` (약어는 풀어서 `aliases`에 추가 — 예: `RPC`, `원격 프로시저 호출`)
- 형식: **한 줄 뜻 → (필요 시) 짧은 부연 → 관련 `[[개념]]` 링크**
- 양식은 `99-템플릿/개발용어-템플릿` 참고
- 다른 노트와는 `[[위키링크]]`로 연결하세요.

## 사용 팁
- 모르는 용어가 뉴스·개념 노트에 나오면 `[[용어]]`로 감싸 두고 여기에 짧게 정리하면, 그래프에서 자동으로 이어집니다.
